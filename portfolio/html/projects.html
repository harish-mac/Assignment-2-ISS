Smart Backup (Bash)

This is a simple backup tool that uses bash scripting to automate the process of backing up files and directories. It allows users to specify source and destination paths, and it can be scheduled to run at regular intervals using cron jobs. The script includes features such as logging and error handling to ensure that backups are performed reliably.

This is a robust backup tool that uses bash. Hence it's very close to the OS and can be integrated easily for file syncing and logging.

Tradeoffs: I used a very simple and straightforward approach to ensure reliability, lacks advanced features like compression.

Reflections: I would like to learn and implement compression to make it faster. GitHub Repo
